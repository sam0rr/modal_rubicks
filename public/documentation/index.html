1. What is Rubicks & Installation

Rubicks is a versatile and customizable JavaScript modal library designed to simplify the process of creating accessible and attractive modals on any website. With Rubicks, you can easily add alert, confirmation, AJAX, and custom content modals with minimal setup.

Installation

To install and use Rubicks, follow these steps:

  1. Download the library files from our page.
    2. Download Rubicks Library
  2. Include the following JavaScript and CSS files in your project:

<link rel="stylesheet" href="path/to/rubicks.css">
<script src="path/to/rubicks.js" type="module"></script>

Replace path/to/ with the correct file paths in your project structure.

2. Modal Types

Alert Modal

The alert modal displays a simple message with an "OK" button to close the modal.


function showAlertModal() {
    Rubicks.showAlert({
        title: "Alert",
        message: "This is an alert modal!",
        okText: "Close",
        callback: () => alert("Alert modal closed")
    });
}
                    

<!-- Button to open the alert modal -->
<button onclick="showAlertModal()">Open Alert Modal</button>

Confirm Modal

The confirm modal presents users with "Yes" and "No" options, and calls a callback function with the user's choice.


function showConfirmModal() {
    Rubicks.showConfirm({
        title: "Confirm",
        message: "Do you confirm this action?",
        okText: "Yes",
        cancelText: "No",
        callback: (result) => alert(result ? "Confirmed" : "Canceled")
    });
}
                    

<!-- Button to open the confirm modal -->
<button onclick="showConfirmModal()">Open Confirm Modal</button>

AJAX Modal

This modal type fetches and displays HTML content from a specified URL, useful for dynamically loading information.


function showAjaxModal() {
    Rubicks.showAjax({
        title: "AJAX Content",
        url: "./ajax.html",
        okText: "Close"
    });
}
                    

<!-- Button to open the ajax modal -->
<button onclick="showAjaxModal()">Open Ajax Modal</button>
